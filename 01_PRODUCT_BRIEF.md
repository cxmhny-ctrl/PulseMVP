# Pulse Product Brief

## Product Name
Pulse

## Core Value Proposition
Pulse is an ADHD task companion that proactively breaks paralysis with ambient nudges, voice, and zero-friction interactions without making the user open an app.

## Product Category
Ambient task-initiation support system for ADHD users.

Pulse should not be framed as another planner, calendar, to-do list, or productivity dashboard. It should be framed as a stuck-to-action engine.

## Target Audiences

### Primary
Adults with ADHD who struggle with task initiation, avoidance, overwhelm, or executive dysfunction.

### Secondary
Families managing ADHD, especially parents supporting kids or teens.

### Professional
ADHD therapists, coaches, and clinicians looking for client support tools and non-clinical progress visibility.

## Core Product Claim
Pulse helps users start when they are stuck, without requiring them to remember, open, organize, type, or maintain another app.

## Strategic Insight
Most productivity apps assume the user can self-initiate. Pulse assumes the user often cannot.

Traditional products ask the user to:

- open the app
- enter tasks
- organize projects
- maintain hierarchy
- remember what matters
- initiate when overwhelmed

Pulse instead does this:

```text
Notice likely stuckness → reach the user → offer one tiny action
```

## Killer Feature
Ambient Stuck Detection and Intervention.

Pulse monitors agreed-upon contextual and behavioral cues, identifies likely paralysis moments, and reaches the user through the lowest-friction enabled channel:

- widget pulse
- notification
- SMS
- voice shortcut
- family/professional support, later

The user can respond with one tap, one SMS reply, or one voice command to start Stuck Mode.

## Must-Have Product Traits

- Proactive, not reactive
- One-step engagement
- No typing required
- App mostly invisible
- Voice-first stuck mode
- Local-first privacy
- User-controlled detection
- Low-pressure nudges
- No shame mechanics

## Do Not Build

- Traditional task manager UI as the main experience
- Complex task hierarchies
- Project management workflows
- Shame-based streaks
- Patronizing gamification
- Subscription-only pricing as the primary model
- Hidden passive monitoring
- Medical diagnosis or treatment claims

## Market Positioning
Unlike visual planners, coaching apps, and to-do managers, Pulse does not require the user to open an app. It reaches the user at the moment of likely stuckness and helps them take one tiny next action.

## Competitive Moat
The moat is not task breakdown by AI. A generic chatbot can do that.

The moat is:

1. proactive detection
2. ambient delivery
3. one-tap/voice activation
4. personalization over time
5. SMS/widget presence
6. optional family/professional loops

## Product Promise

```text
You do not have to finish the task.
You only have to start the smallest next step.
```
